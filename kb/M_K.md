# V60 阿黛尔（AdeleX-10）全局质量矩阵与刚度矩阵

> 基于 V60 4+1 垂起固定翼无人机的具体结构，说明各组件 M/K 如何获得、如何组装耦合、以及激励如何传递。

---

## 1. 任务理解

**目标**：计算 V60 无人机结构固有频率，评估与电机激励频率的避频裕度，避免共振。

**输入**：几何模型（CAD/STL）、材料参数（E,ν,ρ）、质量分布（电机/电池/飞控位置）、连接关系（螺丝夹紧/绑带/RBE3）、电机转速范围。

**方法**：第三方 FEM 求解器（CalculiX）自动完成 K/M 组装 + 特征值求解，用户不手写矩阵公式。输入描述物理模型的 `.inp` 文件，输出固有频率和振型。

**输出**：Campbell 图（激励频率 vs 固有频率）、避频裕度 = |f_exc - f_nat| / f_nat ≥ 10–20%。

**验证**：M-01 悬臂梁（vs FEniCS SLEPc）、M-02 方板（vs NAFEMS FV52）等验证后，确保工具链正确后再用于 V60 全机模型。

## 3. 每个组件的 M 和 K 是怎么来的——不需要你手动组装

### 3.1 不是"一个组件给一个大矩阵块"

有限元软件（ANSYS / Nastran / Abaqus）的工作流程是**自底向上**的：

```
几何模型（CAD / STL）
    ↓ 网格划分
单元（Elements）：每个单元有自己的 M_e (局部) 和 K_e (局部)
    ↓ 软件自动组装（Direct Stiffness Method）
全局 M 和 K（n × n，n = 节点数 × 6 DOF）
```

**你不需要手动组装**。软件做的事情是：

1. 对每个单元，在局部坐标系下计算 $M_e$ 和 $K_e$
2. 将单元局部自由度映射到全局节点编号
3. 迭加到全局矩阵对应位置

### 3.2 不同类型单元的 $M_e$ 和 $K_e$ 长什么样

**V60 使用的单元类型：**

#### (a) 壳单元（Shell）——机身、机翼蒙皮、平尾

用于 3D 打印的薄壁结构（厚度远小于另外两维）。

每个壳单元节点有 **6 个 DOF**（3 平移 + 3 转动），以 4 节点四边形壳为例：

- $K_e$：$24 \times 24$ 矩阵，由膜刚度（拉伸/剪切）+ 弯曲刚度（D 矩阵，正比于 $t^3$）叠加
- $M_e$：$24 \times 24$（一致质量）或对角阵（集中质量）

V60 的 3D 打印件壁厚 ~1 mm，材料为 ABS（E ≈ 2 GPa, ν ≈ 0.35）或 PLA+（E ≈ 3.5 GPa）。

#### (b) 梁单元（Beam）——碳方管翼梁、臂管

碳纤维方管 6×6 mm，壁厚 ~1 mm，E ≈ 50–70 GPa。

两节点梁单元，每节点 6 DOF：

- $K_e$：$12 \times 12$，包含轴向刚度 $EA/L$、弯曲刚度 $EI/L^3$、扭转刚度 $GJ/L$ 的贡献
- $M_e$：$12 \times 12$（一致质量）或 $2 \times 2$ 块对角（集中质量）

#### (c) 集中质量（Concentrated Mass）——电机、电池、飞控、FPV

这些不是"单元"，而是在特定节点上直接附加质量：

- 一个集中质量定义为：质量 $m$ + 转动惯量 $I_{xx}, I_{yy}, I_{zz}$（6×6 对角块）
- 通过 **RBE3**（柔性插值约束）分配到安装面的多个节点

```
电机集中质量的施加方式：

    质量点 M_motor (CG位置，6 DOF)
        │
    RBE3（权重分配）
        │
   ┌────┼────┐
  节点1  节点2  节点3  节点4   ← 电机安装面的 4 个螺栓孔节点
```

**RBE3 的作用**：质量点运动时，力/惯性按权重"分摊"到安装面节点上，但不增加局部刚度（这与 RBE2 不同）。

## 4. 全局 M 和 K 的块结构——以 V60 的实际节点分组画出来

### 4.1 按节点自由度分组

假设 FEM 模型将 V60 的节点按物理归属分为以下组：

```
F   = 机身节点 (Fuselage nodes)             — 约 5000–15000 DOF
WL  = 左翼节点 (Wing Left)                  — 约 2000–8000 DOF
WR  = 右翼节点 (Wing Right)                 — 约 2000–8000 DOF
T   = 尾翼节点 (Tail / Stabilizer)          — 约 1000–4000 DOF
B   = 碳管节点 (Carbon tubes)               — 约 500–2000 DOF
      ├ B_spar = 翼梁碳管 (6×6×500, 横穿机身, 左右套机翼)
      └ B_tube = 纵向碳管×2 (6×6×410, 纵穿机身, 前后装电机, 后端套尾翼)
M   = 集中质量节点 (Motors/Battery/FC等)    — 每个 6 DOF, 约 10–15 个点
```

### 4.3 非对角块的非零条件——"耦合"到底发生在哪

| 耦合对 | K 耦合？ | 为什么 | M 耦合？ | 为什么 |
|--------|---------|--------|---------|--------|
| **机身 ↔ 机翼** | ✅ 有 | 翼梁碳管（6×6×500）穿过机身中央段，左右端分别套入左右机翼，旋钮（knob）锁紧——翼梁节点同时属于机身和机翼单元 | ✅ 有（一致质量）或仅翼梁共享节点 | 翼梁节点跨越机身-机翼边界 |
| **机身 ↔ 纵向碳管** | ✅ 有 | 两根纵向碳管（6×6×410）穿过机身，由 Beamfix 夹紧座 + 螺丝 M2.2×9.5 压紧——夹紧段共节点 | ✅ 通过夹紧位置的共享节点 | 碳管被夹紧段的节点与机身 Beamfix 节点共享 |
| **纵向碳管 ↔ 升力电机** | ✅ 有 | 升力电机座（AAX10.001.Motormount）螺丝 M2×16 固定在碳管前端/后端——共节点 | ✅ 通过 RBE3 分配 | 升力电机集中质量通过 RBE3 连接到电机座螺栓孔节点 |
| **纵向碳管 ↔ 尾翼** | ✅ 有 | 平尾套入两根纵向碳管后端直到止动位，两侧螺丝拧紧——共节点约束 | ✅ 有（一致质量） | 尾翼根部节点与碳管后端节点共享 |
| **机身 ↔ 尾翼** | ❌ **无直接耦合** | 机身和平尾之间只有碳管连接，没有直接物理接触。传力路径：尾翼 → 碳管 → 机身（间接） | ❌ 零 | 无共享节点 |
| **机身 ↔ 推进电机** | ✅ 有 | 推进电机座（MFS.Motormount19）螺丝 M3×8 固定在机身尾部——共节点 | ✅ 通过 RBE3 | 推进电机集中质量通过 RBE3 连接到机身尾部节点 |
| **机身 ↔ 电池** | ❌ 无直接刚度耦合 | 电池仅用绑带固定，不是刚性连接 | ⚠️ 弱耦合 | 可在电池 CG 与机身电池仓节点间加软弹簧模拟 |
| **机翼 ↔ 副翼舵机** | ✅ 有 | 舵机螺丝固定在翼面 servo well 中——共享节点 | ✅ 集中质量分配到舵机安装节点 |
| **左翼 ↔ 右翼** | ❌ **几乎为零** | 左右翼通过翼梁分别插入机身——间接路径是 **左翼→翼梁→机身←翼梁←右翼**，左右翼无直接连接 | ❌ 零 | 没有共享节点 |

### 4.4 关键认识

> - **K 的非对角块就是"物理连接关系图"的矩阵表达**——哪个组件和哪个组件通过螺丝、夹紧、胶接等方式连在一起，对应的 K 子块就非零。
> - **M 的非对角块在集中质量假设下几乎为零**——V60 的惯性耦合很小，因为各集中质量（电机、电池、飞控）之间不共享节点，各自通过 RBE3 独立连接到安装结构。
> - **刚度耦合强度**取决于连接方式：螺丝夹紧 ≈ 刚性共节点，绑带固定 ≈ 弱约束（可忽略或简化为软弹簧）。

## 5. 组装过程的具体例子：机身-纵向碳管-尾翼-升力电机四体耦合

### 5.1 物理连接关系（正确拓扑）

```
                    翼梁碳管 (B_spar, 6×6×500, 横向)
        左翼 (WL) ←══════════╤══════════→ 右翼 (WR)
                             │
                         机身 (F)
                             │
            ┌────────────────┼────────────────┐
            │                │                │
      纵向碳管_L      机身 Beamfix      纵向碳管_R
      (B_tube_L)     夹紧两管          (B_tube_R)
      6×6×410         │               6×6×410
            │                │                │
    前端 ←──┴──→ 后端   (机身内部)  前端 ←──┴──→ 后端
      │          │                    │          │
   升力电机Lf  升力电机Lr         升力电机Rf  升力电机Rr
      │          │                    │          │
      │     ┌────┴────┐              │     ┌────┴────┐
      │     │  尾翼T  │              │     │  尾翼T  │
      └─────┤ (套在管 │──────────────┘─────┤ (套在管 │
            │  后端)  │                     │  后端)  │
            └─────────┘                     └─────────┘
```

**核心认识**：
- **翼梁碳管**（横向）和**纵向碳管**（纵向）是两套独立的承力骨架，都在机身内部被夹紧/穿过，但彼此不直接连接
- 尾翼只与纵向碳管后端连接，与机身无直接接触
- 升力电机全部安装在纵向碳管的两端，不与机翼直接连接
- 推进电机直接安装在机身尾部

### 5.2 未组装前——四个独立的子结构 K 矩阵

```
    F:                 B_tube:              T:                Motor:
  ┌──────┐           ┌──────┐           ┌──────┐           ┌──────┐
  │ K_F  │           │K_tube│           │ K_T  │           │K_mtr │
  └──────┘           └──────┘           └──────┘           └──────┘
 (壳单元组装)       (梁单元组装,        (壳单元组装,       (集中质量,
                    碳管自身弯曲/扭转)   平尾弯曲/扭转)     RBE3分配)
```

### 5.3 组装后——统一的全局 K（以机身 + 一根纵向碳管 + 尾翼 + 一个电机为例）

```
组装 = 共享节点处的 DOF 合并

Beamfix夹紧 → 碳管被夹段的节点 = 机身 Beamfix 节点的 DOF
尾翼套入碳管后端 → 尾翼根部节点 = 碳管后端节点的 DOF
电机座固定在碳管端部 → 电机座节点 = 碳管端部节点的 DOF

              F          B_tube        T         Motor
         ┌─────────┬─────────────┬─────────┬─────────┐
    F    │  K_F    │ K_F,B(夹紧) │   0     │   0     │
         │         │             │(无直接   │         │
         │         │             │ 连接)   │         │
         ├─────────┼─────────────┼─────────┼─────────┤
  B_tube │ (对称)  │   K_tube    │K_B,T    │ K_B,M   │
         │         │             │(尾翼套在│(电机座  │
         │         │             │ 管后端) │ 连接)   │
         ├─────────┼─────────────┼─────────┼─────────┤
    T    │ (对称)  │   (对称)    │  K_T    │   0     │
         │         │             │         │         │
         ├─────────┼─────────────┼─────────┼─────────┤
  Motor  │ (对称)  │   (对称)    │ (对称)  │ K_mtr   │
         │         │             │         │ (RBE3)  │
         └─────────┴─────────────┴─────────┴─────────┘
```

**注意到 K_F,T = 0**——机身和尾翼的唯一传力路径是 **机身 → Beamfix → 碳管弯曲 → 尾翼**，这条路径比"直接连接"更柔，尾翼模态频率会因此偏低。

### 5.4 这个组装过程在软件中的实际操作

1. 在 CAD 中，机身、碳管、尾翼、电机座是**独立的零件/装配体**
2. 网格划分时，在连接面上生成**共节点**（合并重合节点的 ID）：
   - Beamfix 夹紧面 → 合并机身 Beamfix 节点与碳管对应节点
   - 碳管后端 → 合并碳管后端节点与尾翼根部节点
   - 电机座安装面 → 合并碳管端部节点与电机座节点
3. 软件遍历所有单元，对每个单元调用 $K_e$ 和 $M_e$ 计算子程序
4. 根据单元的节点连接关系（connectivity），将 $K_e(i,j)$ 迭加到 $K(node_i, node_j)$
5. 最终得到稀疏、对称、带状的 $n \times n$ 全局矩阵

**你不写任何矩阵组装代码**——你只需要：
- 正确建模几何和材料
- 正确设置连接关系（共节点 / RBE2 / RBE3 / 接触对）
- 正确施加质量分布（集中质量的位置和大小）

## 6. 激励如何传递——从螺旋桨到结构振动的数学表达

### 6.1 激励的物理来源

V60 有 **两套激励源同时或交替工作**：

| 工况 | 工作电机 | 激励频率 |
|------|---------|---------|
| **悬停/起降** (QHOVER) | 4 个升力电机 | 4 组相同/相近的 BPF（取决于各电机转速差） |
| **巡航** (FWBA) | 1 个推进电机（升力电机停转） | 1 组 BPF |
| **过渡** (Transition) | 5 个电机同时工作 | 升力 BPF 递减 + 推进 BPF 递增——频谱最宽 |

### 6.2 单个电机的激励力

以升力电机为例（EMax 2004, 1600kv, 6S ≈ 22.2V）：

- 空载转速 ≈ $1600 \times 22.2 \approx 35500$ RPM（理论值）
- 带桨（Gemfan 5126-2, 2 叶）工作转速：约 4000–8000 RPM（悬停工况）
- **桨叶通过频率（BPF）**：

$$
f_{BPF} = \frac{N_b \cdot \Omega}{60}
$$

其中 $N_b = 2$（桨叶数），$\Omega$ 为转速（RPM）。

| 转速 (RPM) | 1P (不平衡, Hz) | 2P (BPF, Hz) | 4P (Hz) |
|-----------|----------------|-------------|--------|
| 4000 | 66.7 | 133.3 | 266.7 |
| 6000 | 100 | 200 | 400 |
| 8000 | 133.3 | 266.7 | 533.3 |

### 6.3 激励力的数学模型

电机-桨系统的激励力是一个多频简谐力的叠加：

$$
\mathbf{F}_{\text{ext}}(t) = \sum_{k=1}^{N_{\text{harmonics}}} \mathbf{f}_k \cdot e^{i(2\pi k f_{1P} t + \phi_k)}
$$

或者写成实数形式：

$$
\mathbf{F}_{\text{ext}}(t) = \sum_{k=1}^{N_{\text{harmonics}}} \mathbf{f}_k \cdot \cos(2\pi k f_{1P} t + \phi_k)
$$

| 符号 | 含义 | V60 中的来源 |
|------|------|-------------|
| $f_{1P}$ | 基频 = $\Omega/60$ | 转子不平衡（1 阶） |
| $k=2$ | 2 倍频 = BPF | 两叶桨的气动通过频率（最强激励） |
| $k=4,6,\dots$ | 高阶谐波 | 桨叶气动力的高阶分量 |
| $\mathbf{f}_k$ | 第 $k$ 阶激励力的空间分布向量（$n \times 1$） | 由气动力/不平衡力在电机安装节点上的分量决定 |
| $\phi_k$ | 第 $k$ 阶的相位角 | 各电机间的相位差（取决于旋向和同步关系） |

### 6.4 激励力在全局方程中的位置

$\mathbf{f}_k$ 是一个 $n$ 维向量，**非零元素仅出现在电机安装相关的 DOF 上**：

**升力电机**（4 个）——激励力施加在**纵向碳管端部电机座节点**：

$$
\mathbf{f}_k^{\text{(lift)}} = \begin{bmatrix}
0 \\ \vdots \\ 0 \\
f_x \\ f_y \\ f_z \\ M_x \\ M_y \\ M_z \\
0 \\ \vdots \\ 0
\end{bmatrix}
\begin{matrix}
\leftarrow \text{机身 DOF（零）} \\
\\
\leftarrow \text{纵向碳管端部 DOF（非零）} \\
\\
\leftarrow \text{尾翼 DOF（零）}
\end{matrix}
$$

**推进电机**（1 个）——激励力施加在**机身尾部电机座节点**：

$$
\mathbf{f}_k^{\text{(push)}} = \begin{bmatrix}
0 \\ \vdots \\ 0 \\
f_x \\ f_y \\ f_z \\ M_x \\ M_y \\ M_z \\
0 \\ \vdots \\ 0
\end{bmatrix}
\begin{matrix}
\leftarrow \text{纵向碳管 DOF（零）} \\
\\
\leftarrow \text{机身尾部 DOF（非零）} \\
\\
\leftarrow \text{尾翼 DOF（零）}
\end{matrix}
$$

**传递路径**：

```
升力电机激励路径（4条，通过纵向碳管传至机身）：
  桨叶气动力/不平衡力
      ↓
  电机安装座（AAX10.001.Motormount，螺丝固定在纵向碳管端部）
      ↓
  纵向碳管（弯曲 + 扭转振动，6×6×410 碳管为传力主体）
      ↓ Beamfix 夹紧节点（机身内）
  机身（整体振动）

推进电机激励路径（1条，直接传至机身）：
  桨叶气动力/不平衡力
      ↓
  推进电机座（MFS.Motormount19，螺丝 M3×8 固定在机身尾部）
      ↓
  机身尾部 → 机身整体振动

共用末段：
  机身振动
      ↓
  FPV 云台 / 飞控 IMU（振动敏感设备）
      ↓
  机身 → 翼梁碳管 → 机翼（机翼也被带动振动）
  机身 → Beamfix → 纵向碳管 → 尾翼（尾翼也被带动振动）
```

> **关键区别**：升力电机不直接激励机翼。升力电机 → 碳管 → Beamfix → 机身 → 翼梁 → 机翼 是一条**间接传递路径**，振动能量经过多个界面衰减后才到达机翼。同理，推进电机也不直接激励尾翼——尾翼通过碳管后端感受到来自推进电机→机身→碳管的间接激励。

### 6.5 受迫振动的数学形式

$$
\mathbf{M} \ddot{\mathbf{u}} + \mathbf{C} \dot{\mathbf{u}} + \mathbf{K} \mathbf{u} = \mathbf{F}_{\text{ext}}(t)
$$

频域形式（对每个激励频率 $\omega_k = 2\pi k f_{1P}$）：

$$
\left[ -\omega_k^2 \mathbf{M} + i\omega_k \mathbf{C} + \mathbf{K} \right] \mathbf{U}(\omega_k) = \mathbf{F}_k
$$

**共振条件**：当 $\omega_k \approx \omega_n$（某阶固有频率）时，响应幅值急剧增大。

## 7. 具体到 V60 避频分析的关键检查项

### 7.1 需要计算和检查的模态

| 模态 | 预期频率范围 | 关注的激励 | 判定 |
|------|------------|-----------|------|
| 机翼一阶对称弯曲 | 8–20 Hz | 升力桨 1P (67–133 Hz) → **远高于此模态，安全** | ✅ 通常避开 |
| 机翼一阶反对称弯曲 | 10–25 Hz | 同上 | ✅ 通常避开 |
| 机翼一阶扭转 | 30–60 Hz | 升力桨 1P 高转速段 (100–133 Hz)、推进桨 1P | ⚠️ 需校核 |
| 纵向碳管局部弯曲 | 50–150 Hz | 升力桨 BPF (133–267 Hz) | ⚠️ **重点关注** |
| 机身垂向弯曲 | 15–40 Hz | 升力桨 1P、推进桨 1P | ⚠️ 需校核 |
| 尾翼对称/反对称弯曲 | 20–50 Hz | 推进桨 1P (67–133 Hz @ 4000–8000 RPM) | ⚠️ 推进桨激励 |
| 电机安装座局部模态 | 100–300 Hz | BPF 和 4P 的高频分量 | ⚠️ **重点关注** |
| 电池-机身耦合振动 | 10–30 Hz | 低频激励 | ⚠️ 电池质量大，刚度弱 |

### 7.2 V60 特有的结构特点对 M 和 K 的影响

1. **3D 打印薄壁 ABS/PLA+**：刚度偏低（E ≈ 2–3.5 GPa），阻尼偏高（材料自身阻尼有助于抑制共振峰值），FDM 层间方向有刚度折减（约 20–40%），建模时可用各向异性壳单元或折减后的各向同性近似。
2. **两套独立的碳管骨架**：
   - **翼梁碳管**（6×6×500，横向）：承担机翼载荷，穿过机身中央段，是机翼 ↔ 机身耦合的唯一通道
   - **纵向碳管 ×2**（6×6×410，纵向）：承担升力电机载荷 + 尾翼支撑，被机身 Beamfix 夹紧，是 **升力电机 ↔ 机身** 和 **尾翼 ↔ 机身** 的唯一传力通道
   - 两套管在机身内部交叉但不直接连接（分别被不同的夹紧结构固定），耦合通过机身 3D 打印结构传递
   - 碳管 E ≈ 50–70 GPa，整体刚度由"碳管骨架 + 打印蒙皮"并联提供
3. **尾翼通过碳管后端支撑、不接触机身**：尾翼的约束刚度 = 碳管后悬臂段的弯曲刚度，这比"直接插入机身"更柔。尾翼模态频率会因此偏低，需要特别关注与推进电机 BPF 的避频。
4. **绑带固定电池**：刚度耦合弱，电池可能相对机身有独立振动模式，需要作为**弹性连接**而非刚性连接来建模（可用 CBUSH 弹簧单元模拟绑带约束）。
5. **升力电机全部装在碳管两端**：电机质量位于碳管的悬臂端，对碳管的弯曲模态有显著的惯性加载效应（质量越大，碳管弯曲频率越低）。前/后电机与 Beamfix 夹紧点的距离不对称（取决于 CG 配置），导致前后段碳管的局部模态频率有差异。
6. **电机旋向交替**（A CCW, B CW, C CCW, D CW）：各电机的激励力存在相位关系，需考虑**同相激励**（激起对称模态——机身垂向弯曲）和**反相激励**（激起反对称模态——机身扭转/横滚）两种情况。

## 8. 小结：从组件到全局矩阵再到避频分析

```
步骤1: 几何建模 (CAD/STL)
  └→ 定义每个零件的材料、厚度、连接方式

步骤2: 网格划分 (Meshing)
  └→ 壳单元(打印件) + 梁单元(碳管) + 集中质量(电机/电池/飞控)

步骤3: 软件自动组装 M 和 K
  └→ 共节点处理连接面
  └→ RBE3 分配集中质量
  └→ 得到 n×n 的稀疏对称全局矩阵

步骤4: 求解特征值 Kφ = ω²Mφ
  └→ 得到固有频率 f_n 和振型 φ_n

步骤5: 激励力建模
  └→ 每个电机的 BPF = N_b · Ω / 60
  └→ F_ext(t) 施加在电机安装节点上

步骤6: Campbell 图 + 避频裕度
  └→ 激励频率线 vs 固有频率线
  └→ 裕度 = |f_激励 - f_固有| / f_固有 ≥ 10–20%
```

**核心认知**：M 和 K 的块结构反映的就是物理连接关系。**哪个组件和哪个组件螺丝拧在一起，对应的 K 子块就非零**。质量矩阵的非对角耦合主要来自 RBE3 对集中质量的分配，直接惯性耦合（一致质量矩阵的交叉项）在 V60 这种多体装配结构中通常很小。
