# 7/10 周报：无人机模态与避频分析

## 1. 架构初步设计

### 1.1 设计思路

**核心理念**: 用户只描述物理模型（节点、材料、截面、集中质量、约束），不写任何矩阵公式。第三方 FEM 求解器 CalculiX 自动完成刚度矩阵/质量矩阵的组装和特征值求解。

### 1.2 数据流

```
物理模型 (Python API 描述)
  ├── 节点坐标、材料 (E,ν,ρ)
  ├── 梁截面 / 壳厚度 / 实体单元
  ├── 集中质量 (电机、电池)
  └── 约束 / 连接关系
         │
         ▼
  [preprocess/inp_generator.py]  ──→  model.inp  (CalculiX 输入文件)
         │
         ▼
  [CalculiX ccx]                   ← 第三方库，自动完成:
      B32梁 / S4壳 / C3D8实体         K/M 组装 + Lanczos 特征值求解
         │
         ▼
  [solver/calculix.py]           ──→  ModalResult
      .frequencies (Hz)               解析 .dat (频率) + .frd (振型)
      .mode_shapes
         │
         ▼
  [postprocess/]                  ──→  Campbell 图 + 避频裕度报告
```

### 1.3 项目目录

```
drone_fem/
├── materials.py                  # 材料数据库
├── preprocess/
│   └── inp_generator.py          # Python → CalculiX .inp 生成器
├── solver/
│   └── calculix.py               # ccx 调用 + 结果解析
├── postprocess/                  # 后处理 (待开发)
└── data/

m01_validate.py                   # M-01 悬臂梁验证
m02_plate.py                      # M-02 方板验证
```

### 1.4 .inp 模板（CalculiX 输入文件格式）

CalculiX 的 .inp 文件是一种声明式输入格式，用户只需描述"有什么"而不用写"怎么算"。以下是模板结构：

```
** 模型名称

*NODE                              ← 1. 节点坐标
1, 0.0, 0.0, 0.0
2, 1.0, 0.0, 0.0
...

*ELEMENT, TYPE=C3D8, ELSET=Eall   ← 2. 单元定义 (C3D8六面体/B32梁/S4壳)
1, 1, 2, 4, 3, 5, 6, 8, 7
...

*SOLID SECTION, ELSET=Eall, MATERIAL=Mat1  ← 3. 截面属性

*MATERIAL, NAME=Mat1               ← 4. 材料
*ELASTIC
200e9, 0.3                        ← E (Pa), ν
*DENSITY
8000.0                             ← ρ (kg/m³)

*BOUNDARY                          ← 5. 约束 (DOF: 1=ux,2=uy,3=uz,4=rx,5=ry,6=rz)
1, 1, 3                           ← 节点1的 ux=uy=uz=0

*STEP                              ← 6. 分析步
*FREQUENCY
10                                 ← 求前10阶模态
*NODE FILE, U                      ← 输出振型
*END STEP
```

用户只需:
1. 列出节点坐标 (`*NODE`)
2. 定义单元连接关系 (`*ELEMENT`)
3. 指定材料和截面 (`*MATERIAL`, `*SOLID SECTION`)
4. 施加约束 (`*BOUNDARY`)
5. 请求模态阶数 (`*FREQUENCY`)

CalculiX 自动完成: 单元刚度/质量矩阵计算 → 全局 K/M 组装 → Lanczos 特征值求解 → 输出频率+振型。

## 2. 第三方库安装配置

### 2.1 CalculiX (核心 FEM 求解器)

```bash
# 安装 (需要 sudo)
sudo apt install calculix-ccx

# 验证
ccx -v
# 输出: This is Version 2.21
```

CalculiX 是工业级开源 FEM 求解器（GPL 协议），由 Guido Dhondt 开发，被广泛应用于航空航天、汽车等领域。内置丰富的结构单元：

| 单元类型 | CalculiX 关键字 | V60 无人机对应组件 |
|---------|----------------|------------------|
| Timoshenko 梁 (3节点) | `B32` | 碳管骨架 (翼梁、纵向管) |
| MITC4 壳 (4节点) | `S4` | 3D 打印件 (机身、机翼、平尾) |
| 六面体实体 (8节点) | `C3D8` | 通用 3D 模型 |
| 集中质量 | `MASS` | 电机、电池、飞控、FPV |
| 弹簧 | `SPRING2` | 电池绑带 (弱约束) |
| 运动耦合 (RBE3) | `*KINEMATIC COUPLING` | 电机质量分配到安装面 |

自动完成 K/M 矩阵组装和 Lanczos 特征值求解，用户只需编写 `.inp` 输入文件描述物理模型。

### 2.2 Python 依赖


## 3. 完成的算例

### 3.1 M-01: 3D 悬臂梁模态分析

**算例来源**: COMET-FEniCS Modal Analysis demo
> https://comet-fenics.readthedocs.io/en/latest/demo/modal_analysis_dynamics/cantilever_modal.html
> License: Creative Commons Attribution-ShareAlike 4.0 International

**算例参数**:

| 参数 | 值 |
|------|-----|
| 模型 | 3D 悬臂梁 |
| 尺寸 | L=20, B=0.5, H=1 |
| 材料 | E=1×10⁵, ν=0, ρ=1×10⁻³ |
| 单元类型 | C3D8 (8节点六面体) |
| 约束 | 左端面(x=0) ux=uy=uz=0 |
| 参考解 | FEniCS SLEPc (Nx=400, Lagrange P1 3D solid) |

**算例结果** (200×6×11 C3D8, 13200 单元):

| 模态 | CalculiX C3D8 (Hz) | FEniCS 参考 (Hz) | 误差 | Beam 理论 (Hz) |
|------|-------------------|------------------|------|---------------|
| 1 | 2.039 | 2.050 | 0.56% | 2.019 |
| 2 | 4.042 | 4.049 | 0.16% | 4.039 |
| 3 | 12.745 | 12.815 | 0.55% | 12.654 |
| 4 | 25.089 | 25.127 | 0.15% | 25.309 |
| 5 | 35.551 | 35.742 | 0.53% | 35.433 |
| 6 | 66.152 | 66.948 | 1.19% | 70.866 |

**网格收敛性**:

| 网格 | 单元数 | f1 (Hz) | f2 (Hz) | f4 (Hz) | f6 (Hz) |
|------|--------|---------|---------|---------|---------|
| 50×2×3 | 300 | 2.319 | 4.190 | 26.036 | 69.279 |
| 100×3×6 | 1,800 | 2.098 | 4.072 | 25.281 | 67.087 |
| 200×6×11 | 13,200 | 2.039 | 4.042 | 25.089 | 66.152 |
| FEniCS(Nx=400) | — | 2.050 | 4.049 | 25.127 | 66.948 |

**结果分析**:
- ✅ 全部 6 阶模态与 FEniCS SLEPc 参考解误差 < 1.2%，最大误差在 Mode 6 (1.19%)
- ✅ 与梁理论解析解误差 < 7%（Mode 6 的 6.65% 是正常的，3D 实体与梁理论在高阶模态有系统性偏差）
- ✅ 网格从 300 → 13,200 单元，频率单调收敛，证明网格无关性
- ✅ CalculiX C3D8 工作流正确：输入 .inp → 自动 K/M 组装 → Lanczos 特征值求解

### 3.2 M-02: 简支方板模态分析

**算例来源**: NAFEMS FV52 — Simply Supported Solid Square Plate
> NAFEMS Publication TNSB, Rev. 3, "The Standard NAFEMS Benchmarks", 1990
> 公开参考:
> - Altair OptiStruct Verification: https://www.help.altair.com/2021/hwsolvers/os/topics/solvers/os/simply_supported_soid_square_plate_fv52_r.htm
> - Autodesk Fusion Help: https://help.autodesk.com/view/fusion360/ENU/?guid=GUID-1ECFF5CF-9936-4E2F-BFEA-FED0D51E8140

**算例参数**:

| 参数 | 值 |
|------|-----|
| 模型 | 3D 方板 |
| 尺寸 | 10m × 10m × 1m (厚度) |
| 材料 | E=200GPa, ν=0.3, ρ=8000 kg/m³ |
| 单元类型 | C3D8 (8节点六面体) |
| 约束 | 四边简支 (Uz=0, 全厚度) + 角点 ux=uy=0 (消除刚体模态) |
| 参考解 | NAFEMS FV52 (国际标准基准) |

**算例结果** (40×40×8 C3D8, 12800 单元):

| 模态 | 振型 | CalculiX (Hz) | NAFEMS (Hz) | 误差 |
|------|------|--------------|-------------|------|
| 1-3 | 刚体模态 | ~0 | ~0 (参考不计) | — |
| — | 摇摆模态 | 16.34, 23.26 | — (NAFEMS 对称约束抑制) | — |
| 4 | 1阶弯曲 | 44.62 | 45.90 | 2.78% |
| 5 | 1阶扭转 | 108.69 | 109.44 | 0.69% |
| 6 | 1阶扭转 (对称) | 108.71 | 109.44 | 0.67% |
| 7 | 2阶弯曲 | 165.03 | 167.89 | 1.70% |
| 8 | — | 195.12 | 193.59 | 0.79% |
| 9 | — | 206.67 | 206.19 | 0.23% |
| 10 | — | 207.01 | 206.19 | 0.40% |

**网格收敛性**:

| 网格 | 单元数 | Mode 4 (Hz) | Mode 5 (Hz) | Mode 7 (Hz) | Mode 8 (Hz) |
|------|--------|-------------|-------------|-------------|-------------|
| 10×10×2 | 200 | 31.96* | 44.36* | 50.47* | 127.52* |
| 20×20×4 | 1,600 | 22.90* | 32.29* | 45.89* | 112.67* |
| 30×30×6 | 5,400 | 18.80* | 26.69* | 44.96* | 109.74* |
| 40×40×8 | 12,800 | 44.62 | 108.69 | 165.03 | 195.12 |
| NAFEMS | — | 45.90 | 109.44 | 167.89 | 193.59 |

> *粗网格时，摇摆模态与弯曲模态频率接近，数值混叠，模式识别不准确。网格加密后分离。

**结果分析**:
- ✅ 40×40×8 网格下，全部 7 阶弹性模态与 NAFEMS FV52 参考解误差 < 3%，最大误差在 Mode 4 (2.78%)
- ✅ 高频模态 (Mode 9-10) 误差 < 0.5%
- ✅ 扭转模态对 (Mode 5-6) 频率一致 (108.69 ≈ 108.71 Hz)，验证对称性
- ✅ 网格收敛趋势正确，更密网格可进一步降低 Mode 4 误差
- 注：Mode 1-3 为刚体模态 (~0 Hz)；简支板还有两个摇摆模态 (~16 Hz, ~23 Hz，板绕边缘转动)，NAFEMS 通过四分之一对称模型抑制。但对称约束会同时消除反对称扭转模态 (Mode 5-6)，因此全模型保留摇摆模态是正确做法
